## 简介

该系统的诞生的是为了想要写博客同时更新到微信小程序，当然后续可能会扩展更多功能。 
如果您只想搭建一个博客系统，那推荐您使用<a href="https://github.com/ruibaby/halo">halo</a>就能满足您的需求，该系统也是使用halo搭建

轻快，简洁，功能强大，使用 Java 开发的博客系统。
